How it works
  1. A visitor clicks Buy Course on the cinematography page.
  2. They enter their name & email — no card details on this site.
  3. The server calls the Areeba API to create a secure checkout session.
  4. The visitor is redirected to Areeba's hosted payment page to enter card details.
  5. After payment, Areeba redirects back and the sale is recorded automatically.
